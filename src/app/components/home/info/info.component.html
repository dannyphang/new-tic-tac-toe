Welcome to new Tic Tac Toe game!


This is a simple Tic Tac Toe game. You can ONLY play with your friend with current version.

This Tic Tac Toe is same as the classic Tic Tac Toe. The winning condition is to get 3 same shapes in a row, column or diagonal.

But the interesting part is each player can only place up to 3 shapes in the game board. After you placed the THIRD, the FIRST shape will be faded to indicate that the shape will be DISAPPEAR from the board at the next turn. So, the board will always have 3 shapes from each player.

Enjoy!
